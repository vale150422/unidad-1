# Proyecto Final 

Este proyecto es una aplicación con un backend en Django REST Framework y un frontend en Tkinter. Permite gestionar datos mediante una API y una interfaz gráfica siguiendo la arquitectura MVC.

## Tecnologías usadas
- Python 3
- Tkinter
- Django
- Django REST Framework
- SQLite

## Estructura del proyecto

proyecto final/
├── backend/
│ ├── api/
│ ├── project/
│ ├── db.sqlite3
│ └── manage.py
└── frontend/
├── controladores/
├── modelos/
├── vistas/
└── main.py
